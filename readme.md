Unit tests
==========

The `rackunit` tests use the submodule feature new in Racket 5.3. Tests are in
`(module+ test ..)`. You can run the tests for a `foo.rkt` file with `raco
test foo.rkt`. You can run tests for all files using `raco test ./`.
